# Sujet Maison

Ce module contient une implémentation de la fonction partition d'une liste en se basant sur le paramètre taille, pour installer ce module en Production et il suffit d'ajouter cette dépendance dans le fichier POM.

## Getting Started


### Prerequis


```
Java 8
Maven 3.5.3
```

### Installation et Importation du projet


```
Importer le projet dans Eclipse ou dans Spring STS IDE comme un projet maven
```


## Vérifier et exécuter les cas de Test


```
mvn verify
mvn test
```

### Installation et packaging

Pour lancer l'installation et le packaging

```
mvn clean install package
```
